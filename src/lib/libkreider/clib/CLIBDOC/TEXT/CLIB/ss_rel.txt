_ss_rel - release a pending signal
.eb
.sh SYNOPSIS
.bb
int _ss_rel(path)
int path;

.sh DESCRIPTION
.db
.bo
SS_REL
cancels the signal to be sent from a device on data ready. The function
.bo
SS_SIG
enables a device to send a signal to a process when data is available
on the device. If an error occurs, the function returns the value -1 and the
appropriate error value is placed in the global variable
.ul
errno.
.ed
.sh NOTE
.db
The signal request is also canceled when the issuing process dies or closes
the path to the device. This feature exists only on
.bo
SCF
devices.
