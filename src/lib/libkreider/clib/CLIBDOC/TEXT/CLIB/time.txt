.so header
.tl /TIME(3C)/Kreider OS-9 C Library/TIME(3C)/
.sh NAME
.bb
time - Get the time
.eb
.sh SYNOPSIS
.bb
long time ((long *) 0)
long time (tloc)
long *tloc;
.eb
.sh DESCRIPTION
.db
.bo
TIME
returns the value of time in seconds since 00:00:00 GMT, January 1, 1970.
.sp
If
.ul
tloc (taken as an integer) is non-zero, the return value is also stored in
the location to which
.ul
tloc points.
.sp
Upon successful completion,
.bo
TIME
returns the value of time. No error is
possible here as
.bo
TIME
always returns a value.
.sp
The value returned is suited for use with the
.bo
CTIME
function.
.ed
.sh EXAMPLES
.bb
To get the system time value:

long curr_time;
curr_time = time ((long *)0);

or

long curr_time;
time (&curr_time);
.eb
.sh SEE ALSO
.bb
ctime(), o2utime(), u2otime().
.eb
