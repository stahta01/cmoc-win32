.db
.bo
DEVTYP
returns an integer number corresponding to the device type as
defined by OS9.
.ul
PN
is the OS9 path number opened to the device to check.
.PP
The following are the device types and returned values:
.PP
.in +5
0 = SCF
1 = RBF
2 = PIPE
3 = SBF
.PP
.in -5

ISATTY
functions in the same manner as
.ul
devtype,
but returns
.ul
TRUE
if the
device is an
.bo
SCF
type and
.ul
FALSE
if it is not.
.ed

.db

Level II device window since they are defined as SCF type devices.
