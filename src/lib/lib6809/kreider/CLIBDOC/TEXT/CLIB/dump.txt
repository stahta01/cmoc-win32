_dump - Dumps memory to standard output.
.eb
.sh SYNOPSIS
.bb
void _dump(s, addr, count)
int count;
char *s, *addr;
.eb
.sh DESCRIPTION
.db
.bo
DUMP
is used mostly as a debugging function. It prints the title
.ul
s
and then,
starting at the memory address pointed to by
.ul
addr,
dumps at the most
.ul
count
bytes to standard output.
.ed
.sh NOTE
.db
Care must be taken to insure the variable
.ul
count
is not too large, else the
memory dump will take a very long time. Also, the dump is formatted similar
to the standard OS9 dump utility.
.ed
