.sh NAME
.bb
sleep - stop execution for a time
.eb
.sh SYNOPSIS
.bb
unsigned sleep (seconds);
unsigned seconds;
.eb
.sh DESCRIPTION
.db
The current process is stopped for the specified amount of time.  If
.ul
seconds
is zero, the process will sleep for one time-slice (tick).
.sp
The actual suspension time may be different than that requested if a
wake-up signal is received before the sleep timer ends.
