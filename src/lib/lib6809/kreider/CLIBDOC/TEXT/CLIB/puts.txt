.sh DESCRIPTION
.db
The
.bo
PUTS
subroutine copies the null-terminated string
.ul
s
to the standard output stream
.ul
stdout
and appends a new line character.
.PP
The
.bo
FPUTS
subroutine copies the null-terminated string
.ul
s
to the named output
.ul
fp.
.PP
Neither routine copies the terminal null character.
.ed
.sh CAVEATS
.db
The
.bo
PUTS
subroutine appends a new line, while
.bo
FPUTS
does not.  This inconsistency of the new-line being appended by
.bo
PUTS
and not by
.bo
FPUTS
is dictated by history and the desire for compatibility.
