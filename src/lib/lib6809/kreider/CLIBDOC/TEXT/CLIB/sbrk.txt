.sh NAME
.bb
sbrk, ibrk - request additional working memory
.eb
.sh SYNOPSIS
.bb
char *sbrk(increase)
char *ibrk(increase)
.sp
int increase;
.eb
.sh DESCRIPTION
.db
.bo
SBRK
requests an allocation from free memory and returns a pointer to its
base, if successful.
.sp
.bo
SBRK
requests the system to allocate "new" memory from outside the initial
allocation.
.sp
.bo
IBRK
requests memory from inside the initial memory allocation.
.sp
Users should read the Memory Management section of the "C" programming
Manual for a fuller explanation of the arrangement.
.ed
.sh DIAGNOSTICS
.db
.bo
SBRK
and
.bo
IBRK return a -1 if the requested amount of contiguous memory is
not available
.ed
