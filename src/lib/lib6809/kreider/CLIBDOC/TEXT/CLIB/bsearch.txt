bsearch - binary search function
.eb
.sh SYNOPSIS
.bb
char *bsearch(key, base, nel, width, compar)
char *key, *base;
int nel, width;
int (*compar) ();
.eb
.sh DESCRIPTION
.db
This function performs a binary search on already sorted arrays of strings,
finding the string matching
.ul
key,
in the array of strings, beginning at the memory location
pointed to by
.ul
base.  The array MUST have been previously sorted, in assending order,
according to the comparision function
.ul
compar().
The total number of elements in the array
is contained in
.ul
nel,
and the width of each string in the array, which must
all be the same length, is held in
.ul
width.
.PP
The function
.ul
compar()
is any user supplied function that will return whether
the first argument is greater than, equal to, or less than, the second
argument.
.bo
STRCMP
would be a good choice for string variables.
.PP
.bo
BSEARCH
returns a pointer to the matching string upon success, or
.ul
null.
.ed
