CREATE
creates and opens the file named
.ul
fname.
This call accepts the file
mode and access permissions in the same function and is useful in setting up
user permissions as soon as the file is opened.
.PP
This function returns -1 if the file already exists.
.PP
.bo
OCREAT
performs the same function, except it deletes the old file if it
already exists when
.bo

is called.
